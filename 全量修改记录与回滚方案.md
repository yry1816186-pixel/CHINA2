## 4. 回滚说明

### 当前状态

当前目录已经纳入 Git 管理，可通过提交记录进行真实回滚。

### 推荐回滚方式

1. 查看当前提交历史：

```bash
git log --oneline --decorate -5
```

2. 如果要回退到上一个稳定版本，可先创建保护分支，再执行：

```bash
git switch -c backup/current-work
git switch -c rollback/<日期> <稳定提交哈希>
```

3. 如果只想撤销单次提交，优先使用可追溯方式：

```bash
git revert <提交哈希>
```

### 文件级回退思路

如果只想回退单个功能区，可优先按以下粒度处理：

- 页面回退：
  - `src/app/pages/HomePage.tsx`
  - `src/app/pages/BuildingDetailPage.tsx`
- 数据回退：
  - `src/app/data/buildings.ts`

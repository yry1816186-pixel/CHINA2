# First-Wave Agent Starter Pack

## 目的
本文件是一个**可直接给网页 agent 作为起始包的 first-wave 汇总件**。

它不是最终 handoff 包的替代品，
而是为了让你在当前阶段，已经可以把最核心的 scene 身份、边界、任务、风险、页面排序和交付要求打包成一个足够稳的 starter pack。

## 使用前提
本 starter pack 只适用于：
- `scene_03_structure_primary`
- `scene_06_conservation_boundary`
- `scene_01_orientation_order`

前提仍然是：
- 视频先过 `VIDEO_TO_WEB_HANDOFF_GATE.md`
- 审片先过 `FIRST_WAVE_VIDEO_REVIEW_SCORECARD.md`
- intro 不被当作最终冻结入口资产

## 页面目标
- `page_target`: `first-wave controlled long-scroll page`
- `project_status`: `controlled_first_wave_not_final`
- `preferred_page_entry`: `web/reference-shell/index.contract.html`
- `required_contract_sources`:
  - `web/INTRO_ENTRY_EXECUTION_STRATEGY.md`
  - `web/AGENT_IMPLEMENTATION_BOUNDARY.md`
  - `web/reference-shell/REVIEW_CONTRACT.md`
  - `web/reference-shell/scene.contracts.json`
  - `web/scenes.manifest.json`

## 当前必须保留的页面披露
页面不得删除或弱化这些信息：
- current scene status
- evidence grade
- anchor boundary
- allowed mode
- must-not-become / misreading risk
- unresolved / reviewed_candidate 提示

## 给 agent 的最终说明
你当前不是在做最终全站发布版，
而是在做一个受控 first-wave 长滚动页面。

你的任务不是重新解释项目，
而是：
- 严格服从主表 scene 身份
- 严格服从 review contract
- 严格服从 intro strategy
- 严格保留 truth-boundary disclosure
- 在此基础上，把页面做得更接近参考级作品标准

## 当前结论
如果你现在就要把最核心的一包东西交给 agent，

**当前阶段的 first-wave agent starter pack。**

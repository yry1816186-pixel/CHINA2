# First-Wave Asset Dropzone

把用户生成好的 first-wave 视频直接放到这个目录。

## 当前推荐文件名
### 必放 3 条核心视频
- `scene_03_structure_primary.mp4`
- `scene_06_conservation_boundary.mp4`
- `scene_01_orientation_order.mp4`

### 可选 poster
- `scene_03_structure_primary.jpg` / `png` / `webp`
- `scene_06_conservation_boundary.jpg` / `png` / `webp`
- `scene_01_orientation_order.jpg` / `png` / `webp`

### intro 仅作保守测试
- `scene_00_intro_entry.mp4`
- `scene_00_intro_entry.jpg` / `png` / `webp`

## 命名禁令
- 不使用 `_final`
- 不使用 `ultimate`
- 不使用 `masterpiece`
- 不使用任何会制造“已经冻结最终答案”错觉的后缀

## 重要说明
- 用户把文件放到这里之后，就应该把项目交给 agent。
- 用户不应再手工编辑 manifest、手工拼接 handoff 包、手工判断页面入口。
- intro 文件即使存在，也不得自动升级为最终真实入口资产。

## 当前结论
这个目录是**唯一推荐的 first-wave 资产落点**。
